# Lazygit
1. Press `3`  to go to [[3. Local Branches]] panel
2. Select a branch (using `↓`/`↑`/🖱️)
3. Press `d` to delete the branch

# Basic Terminal
```bash
git branch -d <branch>
```
If the branch is not merged, you can use `-D` instead of `-d` to force delete the branch.
```bash
git branch -D <branch>
```
